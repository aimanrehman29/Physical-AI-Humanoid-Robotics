# فزیکل اے آئی اور انسان نما روبوٹکس — ضروریات

یہ نصاب فزیکل اے آئی کو انسان نما روبوٹس کے تناظر میں متعارف کراتا ہے۔ آپ بنیادی اناٹومی، ROS 2، گیزیبو/آئزک سمیولیشن، وژن-لینگوئج-ایکشن لوپس، اور ایک کیپ اسٹون پائپ لائن سیکھیں گے جو سب کچھ جوڑتی ہے۔ سائٹ Docusaurus پر مبنی ہے؛ چیٹ بوٹ بعد میں سِم/حقیقی سوالات کے لیے شامل کیا جا سکتا ہے۔

## اس میں کیا ہے؟
- **ابواب 1–6**: فزیکل اے آئی تعارف، انسان نما روبوٹکس، ROS 2 بنیادیات، ڈیجیٹل ٹوئن سمیولیشن، VLA، اور کیپ اسٹون۔  
- **کیپ اسٹون**: سمیولیشن میں ایک سادہ اے آئی-روبوٹ پائپ لائن بنائیں: محسوس کریں + منصوبہ بنائیں + محفوظ عمل کریں۔  
- **مواد کنٹرول**: Spec-Kit Plus ٹیمپلیٹس اور اسٹائل گائیڈ تاکہ ابواب ہم آہنگ اور قابلِ جائزہ رہیں۔

## سائٹ کیسے استعمال کریں
- سائیڈبار سے ابواب دیکھیں؛ نیا ہوں تو **فزیکل اے آئی تعارف** سے شروع کریں۔  
- قابلِ عمل کوڈ (Python) اور ڈایاگرامز کا استعمال کریں؛ ہر باب چیک لسٹ اور “اجتناب کی غلطیاں” کے ساتھ ختم ہوتا ہے۔  
- کیپ اسٹون اور مواد کنٹرول صفحات آئندہ کام کی فہرست دیتے ہیں۔

## مقامی طور پر چلانا
```bash
npm install
npm run start
```
ڈیولپ سرور http://localhost:3000/ پر چلتا ہے، ہاٹ ری لوڈ کے ساتھ۔ پروڈکشن کے لیے `npm run build` چلائیں۔
